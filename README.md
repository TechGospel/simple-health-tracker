 Simple Health Tracker

A modern, responsive health tracking web application built with Next.js, TypeScript, and Tailwind CSS.
The app allows users to manage medications, log daily vital signs, and securely persist data per user using browser storage.


Tech Stack
Category	Technology
Framework	Next.js (App Router)
Language	TypeScript
Styling	Tailwind CSS
State	React Hooks
Persistence	Local Storage
